The HTML5 date input is now powered with jQuery Tools cross browser dateinput that can be skinned with CSS. If user is using a browser that supports date inputs the input field's type is changed to text so that the native version is not disturbing us. It is only being displayed if JavaScript is disabled.

You need to place the script block after the HTML code or you can alternatively use jQuery's $(document).ready to execute the script as soon as the webpage's Document Object Model (DOM) has been loaded. Read more about that from the User's Manual.
